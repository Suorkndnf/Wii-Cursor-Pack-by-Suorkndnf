This cursor pack is created by Suorkndnf. 

Included in this pack is a file named "install.inf". 
To install the cursor pack:
1. Locate the file "install.inf".
2. Right-click on the file.
3. Select "Install" from the context menu.

After installation, the mouse style configuration window will open automatically.  
Simply click "OK" to apply the cursor scheme.

Enjoy your new cursors!
